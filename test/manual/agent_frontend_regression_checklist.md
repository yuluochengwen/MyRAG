# Agent 前端回归清单

## 基础可用
- [ ] 页面可正常打开：`/static/agent-demo.html`
- [ ] 工具列表可加载并显示至少 2 个工具
- [ ] 输入框支持 Enter 发送、Shift+Enter 换行

## 会话能力
- [ ] 点击“新建”可创建会话
- [ ] 点击会话项可切换历史消息
- [ ] 点击“清空”仅清空当前会话消息
- [ ] 刷新页面后能恢复最近活跃会话

## 请求参数
- [ ] 修改最大迭代后请求生效
- [ ] 修改温度后请求不报错
- [ ] 修改模型名称后请求不报错（模型存在时可返回）
- [ ] 关闭“显示执行步骤”后不渲染步骤卡片

## 交互状态
- [ ] 请求中发送按钮禁用
- [ ] 请求成功后显示统计（迭代/步骤/耗时）
- [ ] 请求失败后显示错误提示且可继续发送
- [ ] 空输入时有提示且不发请求

## 兼容性
- [ ] 桌面端显示正常
- [ ] 移动端（窄屏）无严重错位
