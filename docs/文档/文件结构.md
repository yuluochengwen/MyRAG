## 文件结构

MyRAG/
├── Backend/              # FastAPI 后端
│   ├── app/
│   │   ├── api/         # 路由层 (KB/Assistant/Conversation)
│   │   ├── core/        # 核心配置 (config/database/dependencies)
│   │   ├── models/      # 数据模型 (ORM + Schemas)
│   │   ├── services/    # 业务逻辑 (Chat/Embedding/VectorStore)
│   │   ├── utils/       # 工具类 (文件解析/文本分块/日志)
│   │   └── websocket/   # WebSocket 管理
│   ├── scripts/         # 数据库初始化脚本
│   └── config.yaml      # 配置文件
├── Frontend/            # 前端静态资源
│   ├── js/             # 业务逻辑
│   ├── css/            # 样式文件
│   └── *.html          # HTML 页面
├── KnowledgeBase/      # 文件存储 (kb_1/files/)
├── VectorDB/           # ChromaDB 向量数据库
├── Models/             # 本地模型
│   ├── Embedding/      # 嵌入模型
│   └── LLM/            # 大语言模型
└── logs/               # 日志文件
