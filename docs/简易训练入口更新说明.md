# 简易 LoRA 训练入口 - 更新说明

## 🎯 更新内容

已在模型管理页面添加 **3 个** 显眼的简易训练入口！

---

## 📍 入口位置

### 1️⃣ **页面顶部横幅**（最醒目）

**位置**：模型管理页面，标题上方  
**样式**：紫色→蓝色→靛蓝色渐变背景  
**内容**：

```
┌─────────────────────────────────────────────────────────────┐
│  ✨ 简易 LoRA 训练 - 3 步完成                               │
│  上传数据集 → 选择模型 → 开始训练，参数自动优化，无需配置！ │
│                                     [立即开始训练 →]        │
└─────────────────────────────────────────────────────────────┘
```

**特点**：
- 渐变色背景，视觉冲击力强
- 白色大按钮，一键跳转
- 简洁的 3 步说明

---

### 2️⃣ **LoRA 标签页内的训练中心面板**

**位置**：点击"LoRA模型"标签 → 训练中心面板  
**布局**：

```
┌──────────────────────────────────────────────────────────┐
│  🎓 LoRA 模型训练中心                                    │
│  简易训练：一键上传数据集即可训练                        │
│  LLaMA-Factory：零代码专业训练平台                       │
│                                                          │
│  [✨ 简易训练] [状态: 未运行] [▶ LLaMA-Factory]        │
└──────────────────────────────────────────────────────────┘
```

**按钮样式**：
- **简易训练**：紫色→蓝色渐变，带魔法图标 ✨
- **LLaMA-Factory**：绿色按钮
- 两种训练方式并列，功能明确

---

### 3️⃣ **简易训练页面的返回链接**

**位置**：简易训练页面顶部  
**样式**：

```
← 返回模型管理

简易 LoRA 微调
只需上传数据集和选择基座模型，其余自动处理
```

**作用**：方便用户返回模型管理页面查看已训练模型

---

## 🎨 视觉设计

### 颜色方案

| 元素 | 颜色 | 用途 |
|------|------|------|
| 顶部横幅 | 紫色→蓝色→靛蓝色渐变 | 吸引注意 |
| 简易训练按钮 | 紫色→蓝色渐变 | 主要操作 |
| LLaMA-Factory 按钮 | 绿色 | 高级功能 |
| 返回链接 | 灰色→蓝色（hover） | 次要导航 |

### 图标使用

- ✨ **魔法图标**（fa-magic）：简易训练
- 🎓 **学士帽**（fa-graduation-cap）：训练中心
- ◀ **左箭头**（fa-arrow-left）：返回

## 🌐 访问路径

### 方法 1：从模型管理进入

1. 访问 `http://localhost:8000/static/model-management.html`
2. 页面顶部看到紫色横幅
3. 点击"立即开始训练"按钮

### 方法 2：从 LoRA 标签进入

1. 访问模型管理页面
2. 点击"LoRA模型"标签
3. 点击"✨ 简易训练"按钮

### 方法 3：直接访问

直接访问：`http://localhost:8000/static/simple-lora-training.html`
